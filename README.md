# nanoc-handlebars

This provides a [Handlebars](http://handlebarsjs.com/) filter for [nanoc](http://nanoc.ws), using [Handlebars.rb](https://github.com/cowboyd/handlebars.rb).

## Installation

`gem install nanoc-handlebars`

## Usage

```ruby
filter :handlebars
```

This filter takes no options.
